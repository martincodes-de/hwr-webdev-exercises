6. Forms

In HTML there are multiple variants to use forms.
First you have to know that every form that you can use is an kind of an input object.
At first we have to open an form tag with <form>
now to get input from the user we will use the <input> tag.
Inside this tag you can define what type of an input the tag should display.
After finishing the inputtag you will have to close it with </form>
The following section will describe how to use the inputtypes correctly.
  • Text
  • Radio
  • checkbox
  • submit
  • button

Text

To get an input of the user we want to use an inputtag.
For that we first have to open an form tag and than we can represent multiple inputs.
In my example shown below, i first display an label to tell the user i want their
first name. As you can see we just have to use the labeltag and put our text, we
want to display, inside of it. After the Label we use the inputtag and set it to the
"text" type, if you like to, you can give it an id and a name to refer to it later on.

For my example i used two times the tags to get the first and last name of the user.
<form> <label>First name: </label> <input type="text" id="fname" name="fname"><br> <br> <label>Last name: </label> <input type="text" id="lname" name="lname"> </form>



radio

If you want to let the user choose one thing from many than you should use the "radio"
type in the inputtag. In my example i want the user to choose his favorite programming
language. There can the user choose between C#, Javascript and Java. In this case i
placed the label after the inputtag to display it on the right side of the box.

Remember!
The user can choose only one thing inside the whole form(group). <form> <label>Choose your favoriete language: </label><br> <input type="radio" id="c-sharp" name="fav_language"> <label>C#</label><br> <input type="radio" id="javascript" name="fav_language"> <label>Javascript</label><br> <input type="radio" id="java" name="fav_language"> <label>Java</label><br> </form>





Checkbox

The checkbox is kinda similar to the radio type, exept that the user gets the ability to
check multiple boxes. In the example shown you can see that you can check the boxes
as much as you like without taking your check away. This is possible with the
inputtype "checkbox".
