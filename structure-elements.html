How to make it yourself

How to create a division object

To create a division object you have to use the keyword < div > like shown in the code below.

<div>
    <p>This is my first paragraph.</p>
    <p>This is my second paragraph.</p>
    <p>This is my final paragraph.</p>    
</div>


How to create a span object

To create a span object you need to use the keyword < span > like shown in the code below.

<p>This is a paragraph with <span>a little something extra</span> inside it.</p>
