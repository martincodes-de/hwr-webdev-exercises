Hyperlinks


Hyperlinks can be implemented by using a website link and could look like this:

link to website for information on hyperlinks

For this example the code is as follows:

<a href="https://www.schulhomepage.de/webdesign/html-link">link to website for information on hyperlinks</a>

It is also possible to link your own html documents to switch between multiple websites. The code for that could look like this: <a href="main.html">Homepage</a>

For this example the link path can be used without changes, but for a document that is in a different folder you would have to write a different path.
