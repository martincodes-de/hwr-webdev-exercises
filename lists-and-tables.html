Lists

Unordered Lists

Unordered Lists are built inside the <ul> and </ul> tags. The list-elements are built with <li> and </li>.

An example code and list could look like the following: 

    <ul>
        <li>First Element</li>
        <li>Second Element</li>
        <li>Another Element</li>
    </ul>
  • First Element
  • Second Element
  • Another Element

Ordered Lists

Ordered Lists are built inside the <ol> and </ol> tags. The list-elements are also built with <li> and </li>.

An example code and list could look like the following: 

    <ol>
        <li>First Element</li>
        <li>Second Element</li>
        <li>Another Element</li>
    </ol>
  1. First Element
  2. Second Element
  3. Another Element

Tables

Tables are built with the <table> tags. Inside these tags you specify the rows with <tr> ("tr" = "table-row") and the corrisponding cell with <td> ("td" = "table-data"). A header cell is built with <th> ("th" = "table-header").

An example code and table could look like the following: 

    <table>
        <tr>
            <th>Hello</th>
            <th>I am</th>
            <th>the table header</th>
        </tr>
        <tr>
            <td>Hello</td>
            <td>I am</td>
            <td>a table row</td>
        </tr>
        <tr>
            <td>And I am</td>
            <td>another</td>
            <td>table row</td>
        </tr>
    </table>
Hello I am the table header
Hello I am a table row
And i am another table row
