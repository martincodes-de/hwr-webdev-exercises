Introduction to Semantic HTML Elements (main / section / h1)

Semantic HTML elements are used to describe the meaning and structure of the content on a webpage. This makes it easier for search engines and assistive technologies to understand the content of the page.

Block-level Elements (main / section / h2)

Block-level elements create a new block formatting context, and take up the full width of their parent container. Examples include:

  • header
  • nav
  • section
  • article
  • aside

Inline-level Elements (main / section / h3)

Inline-level elements do not create a new block formatting context and only take up as much width as necessary. Examples include:

  • span
  • a
  • strong
  • em
  • img
